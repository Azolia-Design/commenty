Frequently Asked Questions

Not yet found what you are looking for? We’re here to help.

support@commenty.com

My clients are not tech-savvy, can they use Commenty easily?

Commenty interface is super simple and straightforward, which means no need for training or figuring out how to do things.

Do my clients have to install anything before using it?

No downloads, no installations. Commenty is a web-based app, which works well on any browser so your clients can jump in and start using it right away.

How can I set up before sharing the space with my team and clients?

Type in a website URL or upload images, then share a link with anyone to invite them to comment.

Does Commenty work with responsive websites?

Yes it does, you can see your websites in both desktop and mobile viewport

How can I update feedbacks into my team's workflow?

Commenty is connected with other popular management apps like Monday, Jello, etc. You can export the feedbacks into these apps and start planning your work.

Do I need a canvas for every page?

Nope! Each canvas can contain all pages of a website, you can switch to browsing mode and navigate to any page on that website

Does Commenty work with Wordpress/ React/Angular?

Commenty accepts all types of websites.
